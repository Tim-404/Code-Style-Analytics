Coding in Style

This website will analyze source code for the following:

  • Stray Literals (aka "magic numbers")
  • Indentation Consistency
  • Vague Variable Names

Supported languages include:

  • Java

Submit the source code and run the program.

This stray literal finder was made to assist with grading code style. Avoid completely depending on this program as a rubric. It is up to the grader to cross-check the results with the student's source code.
